# GrokThumbnails

Free YouTube tools: thumbnail downloader + video downloader, merged into a
single Next.js site with a shared theme, navigation, and SEO setup.

## Structure

- `/` — YouTube thumbnail downloader (client-side, no API key needed)
- `/youtube-video-downloader` — YouTube video downloader (calls `/api/download`, needs `RAPIDAPI_KEY`)
- `/blog/thumbnail-design-tips-2026` — blog article
- `/blog/thumbnail-copyright-mistakes` — blog article
- `/about`, `/contact`, `/privacy`, `/terms` — trust pages
- `/sitemap.xml`, `/robots.txt` — generated by `app/sitemap.js` and `app/robots.js`

## Setup

```bash
npm install
cp .env.example .env.local   # add your RapidAPI key
npm run dev
```

## Before deploying

1. Set `RAPIDAPI_KEY` in your hosting provider's environment variables (e.g. Vercel → Settings → Environment Variables).
2. Replace `support@grokthumbnails.com` in `app/contact/page.js` and `app/privacy/page.js` with your real inbox.
3. Add a real Open Graph preview image if you want rich social previews.
4. Confirm `https://grokthumbnails.com` is correct everywhere (used in `app/layout.js` metadataBase and `app/sitemap.js` / `app/robots.js`) — update if the domain changes.

## Deploy

Works on Vercel out of the box (`next build` / `next start`). Any Node host
that supports Next.js App Router API routes will work too — a fully static
host will NOT work because `/youtube-video-downloader` depends on the server-side
`/api/download` route.
